Adventures in Umalu

Adventures in Umalu ist ein rogue-like Dungeon-crawler Spiel, geschrieben in Java. Es wurde im Zuge des Sofwarepraktikum-Kurses 2013 des Instituts für Informatik der Freien Universität Berlin von

  • Dirk Braun
  • Paul Kermas
  • Nicolas Loerbroks
  • Markus Penner
  • Christoph Spenke

entwickelt. Adventures of Umalu verwendet die jade-rogue Library von Jeffrey Lund.

Adventures of Umalu besteht aus zwei Spielen:

Adventures in Umalu
- Geh auf ein Abenteuer um den Drachen Ra's al-Ghul aus dem Berge Umalu zu vertreiben.
Running in Umalu
- Geh auf Speedrun in den Karten von Adventures in Umalu. Mache dabei so wenige Schritte in so kurzer Zeit wie möglich.

Es gibt die folgenden Möglichkeiten um Adventures oder Running in Umalu zu spielen:

  • Sourcen laden und kompilieren
  • .jar Datei laden und starten
  • Online spielen

Sourcen laden und kompilieren

  1. Lade die Daten mit git von der Projektseite: 
    git clone https://github.com/26thmeusoc/adventure-of-umalu.git adventure-of-umalu
  2. Wechsel mit deinem Terminalclient in den neuen Projektordner: 
    cd adventure-of-umalu/src
  3. Kompiliere das Projekt: 
    javac rogue/Rogue.java
  4. Starte das Programm (Adventures in Umalu) 
    java rogue/Rogue
  5. oder: Starte Running in Umalu: 
    java rogue/Rogue speedrun

.jar Datei laden und starten

  1. Lade die .jar Datei herunter: Herunterladen
  2. Öffne deinen Terminalclient und wechsel in den Ordner mit der .jar Datei
  3. Starte Adventure of Umalu 
    java -jar adventure-of-umalu.jar
  4. oder: Starte Running in Umalu: 
    java -jar adventure-of-umalu.jar speedrun

Online spielen

  1. Stelle sicher dass dein Browser Java unterstützt und dein Java Webstart aktiviert ist
  2. Folge einem der beiden Links:
    • Adventures in Umalu spielen
    • Running in Umalu spielen
