Adventure at Umalu ist ein rogue-like Dungeon-crawler Spiel, geschrieben in Java. Es wurde im Zuge des Sofwarepraktikum-Kurses 2013 des Instituts für Informatik der Freien Universität Berlin von

  • Dirk Braun
  • Paul Kermas
  • Nicolas Loerbroks
  • Markus Penner
  • Christoph Spenke

entwickelt. Adventure at Umalu verwendet die jade-rogue Library von Jeffrey Lund.

Adventure at Umalu besteht aus zwei Spielen:

Adventure at Umalu
- Geh auf ein Abenteuer um den Drachen Ra's al-Ghul aus dem Berge Umalu zu vertreiben.
Running in Umalu
- Geh auf Geschwindigkeitsjagd in den Karten von Adventure at Umalu. Mache dabei so wenige Schritte in so kurzer Zeit wie möglich.

Features

  • Ein Field of View
  • Inventar
  • Viele Gegenstände die man finden kann

Adventure at Umalu spielen

Es gibt die folgenden Möglichkeiten um Adventures oder Running in Umalu zu spielen:

  • .jar Datei laden und starten
  • Sourcen laden und kompilieren (fortgeschritten)

.jar Datei laden und starten

  1. Lade die .jar Datei herunter: Herunterladen
  2. Öffne deinen Terminalclient und wechsel in den Ordner mit der .jar Datei
  3. Starte Adventure at Umalu 
    java -jar adventure-of-umalu.jar
  4. oder: Starte Running in Umalu: 
    java -jar adventure-of-umalu.jar speedrun
  5. Hinweis: Unter MacOS X genügt ein Doppelklick auf die heruntergeladene adventure-of-umalu.jar Datei um Adventure at Umalu spielen zu k¨nnen

Sourcen laden und kompilieren (fortgeschritten)

  1. Lade die Daten mit git von der Projektseite. Entweder durch klicken auf den .zip Button oder mit git: 
    git clone https://github.com/26thmeusoc/adventure-of-umalu.git adventure-of-umalu
  2. Wechsel mit deinem Terminalclient in den neuen Projektordner: 
    cd adventure-of-umalu/src
  3. Kompiliere das Projekt: 
    javac rogue/Rogue.java
  4. Starte das Programm (Adventure at Umalu) 
    java rogue/Rogue
  5. oder: Starte Running in Umalu: 
    java rogue/Rogue speedrun
